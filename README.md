# image-classify
Service in Node JS to classify image with words
## Usage
Build image, run container, show logs
```
docker-compose build && docker-compose up -d && docker-compose logs -f
```
## Endpoints
```
http://[HOST]:[PORT]/image/?url=[IMAGE_URL]
```
